# Core API SDK for PHP
This package is currently work in progress


### Installation
First, you need to add it to your composer.json repositories array.
``` json
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/airlst/sdk-php.git"
    }
]
```
Require the package.
``` bash
composer require airlst/sdk-php
```
